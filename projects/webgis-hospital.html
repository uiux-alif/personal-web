Project Overview

Finding the right hospital
shouldn't be hard.

Residents of Semarang faced a significant challenge: when in need of medical attention, there was no centralized, digital tool to quickly find the nearest suitable hospital based on their specific medical needs. People relied on word-of-mouth, outdated pamphlets, or time-consuming Google searches.

This project set out to solve that problem through a web-based GIS application — combining geographic data from the city of Semarang with an intuitive, user-centered interface that made hospital discovery fast, accurate, and accessible to anyone with a smartphone or computer.

As the lead designer, I was responsible for the full UX process: user research, journey mapping, wireframing, visual design, and the final interactive Figma prototype. The geospatial data layer was built in collaboration with a GIS engineer using SuperMap.

Key Challenges

Problems we solved.

Designing a spatial application for non-technical users presented unique UX challenges that required thoughtful solutions.

01
Map Complexity for Non-GIS Users
Most users had no experience with GIS tools. The map interface had to feel as intuitive as Google Maps — with simplified controls, clear visual hierarchy, and progressive disclosure of spatial data.
02
Medical Data Hierarchy
Hospital information is complex — types, specializations, facilities, hours, capacity. Designing information architecture that surfaced the right details at the right moment required extensive user research.
03
Mobile-First in a Low-Bandwidth Context
Most users would access the tool on mobile during emergencies, often in areas with poor connectivity. The design had to be lightweight, fast, and functional on low-end devices.
04
Spatial Data Integration
Bridging the gap between GIS data layers and a consumer-facing UI required close collaboration with the geospatial engineer to ensure data accuracy while maintaining design integrity.

Features

What's inside the app.

Core features designed to make hospital discovery seamless, accurate, and genuinely useful.

📍
Location-Based Search
Automatically detects the user's location and displays hospitals sorted by proximity using spatial distance calculation.
🏥
Specialization Filter
Filter hospitals by medical specialization — cardiology, neurology, pediatrics, general, and more — so users find exactly what they need.
🗺️
Interactive GIS Map
A fully interactive web map powered by SuperMap with custom-styled markers, clustering, and smooth pan/zoom interactions.
📋
Rich Hospital Cards
Detailed information cards with address, rating, operating hours, bed availability, contact, and direct navigation links.
⚡
Radius Control
Adjustable search radius slider lets users expand or narrow their search area dynamically on the map.
📱
Responsive Design
Fully responsive interface optimized for mobile, tablet, and desktop — ensuring usability in emergency situations from any device.
